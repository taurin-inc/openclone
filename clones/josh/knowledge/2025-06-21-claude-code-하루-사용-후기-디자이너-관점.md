Claude Code를 하루종일 써본 후기 
(디자이너 관점, 기술적 용어 주의)

1. 이렇게 한방에 끝내준다고..? 한큐에 완성해주는 신기한 경험
2. 에러도 스스로 잡아서 다 해결해버리는 신기한 경험
3. 롤백 기능이 없어서 방법을 몰라 헤메게 되는 불편함
4. UI 디테일 잡는 건 그래도 커서의 stagewise를 쓰게 됨
5. 되게 오래걸리지만 결과물이 만족스럽기 때문에 참을만 함
6. 클로드 코드라도 디자인은 여전히 그렇게 썩 잘 하진 않음

결론: IDE로 인터페이스 개선해서 나오면 커서를 이기지 않을까?
