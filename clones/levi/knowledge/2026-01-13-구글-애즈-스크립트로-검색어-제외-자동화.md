# 구글 애즈 스크립트로 검색어 제외 자동화🤖

**작성자:** _keunyoung_  
**날짜:** 2026-01-13

구글 애즈 스크립트로 검색어 제외 자동화🤖

구글 검색광고에서 키워드 확장기능을 활용하면 유효 키워드에 확장 가능성이 높지만, 반대로 불필요한 키워드나 성과가 낮은 검색어도 노출이됩니다. 이런 검색어는 정기적으로 제외가 필요한데, 너무 귀찮아서 구글 애즈 스크립트로 자동화했습니다.

스크립트 구현할 때 N개 검색어, N원 절감을 반영해두면 조금 더 편하게 로그를 볼수 있습니다.

1. 필터 규칙을 설정합니다.
2. 구글 애즈에 반영할 스크립트 구현 (Gemini 모델을 적용해서 키워드 유사성 검증)
3. 스크립트 일정 설정
4. 실행 로그 검토
