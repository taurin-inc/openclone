# sf-required
Manage recurring validation states.

* **Field Input (Required)**: sf-form_input, sf-required
* **Checkbox (Required)**: sf-form_checkbox-field, sf-required
* **Radio (Required)**: sf-form_radio-field, sf-required
* **Select (Required)**: sf-form_input-select, sf-required
* **Date Input (Required)**: sf-form_input-date, is-icon-left-right, sf-required
* **Error Tag (Required)**: sf-form_input-error-wrapper, sf-required
* **Icon on Input Right (Required)**: sf-form-icon-right, sf-required
* **Icon on Input Left (Required)**: sf-form-icon-left, sf-required
* **Icon on Input Right Text Area (Required)**: sf-form-icon-right, is-text-area, sf-required

## sf-checked
Manage recurring checked radio & checkboxes states.

* **Checkbox (Checked)**: sf-form_checkbox-field, sf-checked
* **Radio (Checked)**: sf-form_radio-field, sf-checked

## sf-focus
Manage recurring focusing for button, radio and checkbox states.

* **Radio (Focused)**: sf-form_radio-field, sf-focus
* **Checkbox (Focused)**: sf-form_checkbox-field, sf-focus

## sf-hide
Manage awaiting states.

* **Loader Box (Currently Hidden)**: sf-skeleton, sf-hide

## sf-await
Manage awaiting states.

* **Awaiting Status of Buttons**: sf-button-child, sf-await, sf-button-await-child

## sf-current
Manage current states of progress steps.

* **Progress Steps (Current)**: sf-form_progress-step-text, sf-current, sf-form_progress-step-line

## sf-completed
Manage completed states of progress steps.

* **Progress Steps (Completed)**: sf-form_progress-step-text, sf-completed, sf-form_progress-step-line, sf-form_progress-step-icon-wrapper

* **회사명***

* **추천인 / 추천회사(선택사항)**

* **서비스 제공 플랫폼(모두 기재)***
