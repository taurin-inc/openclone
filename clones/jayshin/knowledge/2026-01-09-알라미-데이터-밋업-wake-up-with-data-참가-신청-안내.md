📣 딜라이트룸 데이터그룹이 주최하는 첫 오프라인 행사, 알라미 데이터 밋업: Wake Up with Data 의 참가 신청을 받습니다. (~1/15)

딜라이트룸은 외부 투자 없이 매년 꾸준히 성장해 온 글로벌 스타트업으로, 알라미(Alarmy)와 다로(DARO) 서비스를 통해 수많은 사용자 데이터를 기반으로 비즈니스를 확장해오고 있습니다.

이번 밋업에서는
👉 알라미·다로의 실제 데이터 활용 사례
👉 데이터 분석 환경과 조직 문화
👉 데이터가 어떻게 성과와 임팩트로 이어지는지
등을 데이터 분석가의 시선에서 솔직하게 공유할 예정입니다.

제품과 비즈니스에 진짜 영향을 주는 데이터 분석에 관심 있는 분, 성장하는 조직에서의 데이터 역할과 고민이 궁금한 분들의 많은 참여를 기다립니다. (딜라이트룸 데이터 분석가들과의 커피챗을 원하시는 분들도 환영!)

※ 참가 신청자 중 추첨을 통해 참석자를 선정하며, 선정 결과는 개별 안내드립니다.
※ 간단한 저녁 식사가 제공됩니다.
