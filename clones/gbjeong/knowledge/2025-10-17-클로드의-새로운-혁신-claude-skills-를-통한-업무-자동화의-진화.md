와.. 업무 자동화의 신세계가 열렸습니다. 이제 나만의 Manus를 만들 수 있습니다. "Claude Skills" 저희 팀은 이 기능으로 갈아탑니다..

1. 이제 나만의 워크플로우를 무한대로 만들 수 있습니다.
지금까지는 mcp 로 누군가 만들어놓은 것이나, manus-genspark 에 이미 구현된 기능을 사용할 수 밖에 없었지만 클로드에게 스킬을 만들어달라고 요청하면 즉시 만들고 필요할 때 자동으로 사용합니다.

2. 에이전트에게 docx 작성, 엑셀 편집 같은 능력을 ‘문서와 코드를 합쳐서’ 제공하는 시스템
스킬은 프롬프트-스크립트-데이터를 담은 모듈형 단위입니다. 이를 통해 특정 조직, 업무, 도메인에 맞춤화된 기능을 손쉽게 만들 수 있습니다. 

3. 모바일에서 클로드 코드급 에이전트를 바로 사용
이번 기능의 대단한 점은 모바일/웹/앱에서 모두 작동한다는 것입니다. 개발자용 mcp 연결은 강력했지만 claude desktop에서만 가능했습니다. 그것도 비개발자는 알기 어려운 json 을 수정해야했죠. "개발자" 탭에 들어있어서 무섭기도 했고요. 하지만 클로드에서 sandbox를 제공해서 이번에는 모든 환경에서 동작합니다.

4. 바로 코드를 쓰고 실행합니다.
skills은 얼마 전 추가된 "code execution and file creation" 설정으 켜야 사용할 수 있습니다. 프롬프트-스크립트-데이터를 직접 클로드가 파일로 쓰고 업데이트하거든요.

조만간 비개발자 쪽에서도 클로드코드급 모먼트가 등장할 것 같습니다.
