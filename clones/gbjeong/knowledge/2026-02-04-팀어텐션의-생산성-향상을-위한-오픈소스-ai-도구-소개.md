팀어텐션은 claude code 를 더 잘 쓰기 위한 제품을 직접 만들어서 쓰고 있습니다. 더 많은 팀이 직접 툴을 만들어 쓰시길 응원합니다! 모두 오픈소스이니 사용해보세요 :)

1. cops - claude code 사용량 통합하고 시각화
~/.claude 디렉토리를 실시간 감시해서 세션별 토큰 소비량, 채팅 기록, 에이전트 워크플로우를 대시보드로 보여줍니다. CLI로 프로젝트 등록하면 Daemon이 백그라운드에서 JSONL 로그를 파싱해서 API 서버로 전송. "이번 달 얼마나 썼지?", "어떤 프로젝트에서 가장 많이 썼지?" 궁금할 때 웹 대시보드에서 바로 확인.
https://lnkd.in/gEfciiRJ

2. nobs whisper - whisper 모델을 로컬에서 돌린다. 돈 낼 필요 없음
바로 다운 받아서 사용 가능합니다. 저도 이제 wispr flow 안 쓰고 이것 씁니다. Apple Silicon Metal 가속으로 빠르고, 음성 데이터가 내 맥북 밖으로 안 나갑니다. 글로벌 핫키 설정하면 어떤 앱에서든 바로 받아쓰기. 한국어, 영어, 일본어, 중국어 지원하고, 커스텀 어휘 등록으로 전문 용어 인식도 가능.
https://lnkd.in/guAq5qXV


3. nobs editor - 터미널에서 파일 열었을 때 가볍고 깔끔하게 볼 수 있는 노션 스타일 에디터
Markdown, JSON, YAML, TOML, XML, CSV 등 13가지 형식 지원. IDE 켜기 귀찮을 때 Cmd+O로 파일 열고, 슬래시 명령어(`/`)로 헤딩, 리스트, 코드블록 빠르게 삽입. 드래그 앤 드롭, Finder 우클릭 메뉴 통합까지. Tauri 기반이라 가볍습니다.
https://lnkd.in/gNVJs9k3


+contents hub - 컨텐츠 소화의 핵심
내가 선택한 채널을 x, youtube, rss 가리지 않고 구독해서 한 판에 요약해줍니다. 현재 Karpathy, Ryan Carson, Every, a16z, Lenny's Podcast, Dwarkesh Patel 등 14개 소스에서 매일 자동 수집. 관심사(lens) 설정하면 AI가 클러스터링해서 1-page 다이제스트로 만들어줍니다. MCP로 Claude Code 연결하면 "이 글 원문 읽어줘", "댓글 반응 어때?" 같은 심층 분석도 가능. 5분 안에 AI 트렌드 캐치.


직접 만들어 쓰는 이유
- 도구에 맞추지 않고 워크플로우에 맞는 도구 만듭니다.
- 만드는 과정 자체가 AI를 더 깊이 이해하는 방법이라고 생각합니다.
- 오픈소스로 공개해서 커뮤니티와 함께 발전하고 싶습니다.
