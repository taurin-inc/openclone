이 작은 앱을 저는 앞으로 항상 켜놓을거에요. Jongwon Choi님이 Claude Code 의 Tasks를 추적하는 앱을 만들어주셨어요.

Claude Code로 복잡한 작업을 시키면 TodoWrite 도구로 할 일 목록을 만들어요.
문제는 이게 터미널 안에 묻혀서 잘 안 보인다는 거죠.

이 앱은 그 태스크들을 플로팅 윈도우로 띄워줍니다.

주요 기능:
- 파일 변경 시 자동 새로고침
- 세션별로 필터링 가능
- ⌘E로 태스크 바로 수정
- ▶ 버튼으로 해당 세션 터미널에서 바로 열기

GitHub: https://lnkd.in/gFNPP24Q

세션 여러 개 돌리시는 분들 진짜 추천드립니다!
