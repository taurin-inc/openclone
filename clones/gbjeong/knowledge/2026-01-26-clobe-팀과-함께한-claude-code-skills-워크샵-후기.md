Clobe 팀과 Claude Code skills 워크샵을 같이 했습니다. 가장 놀라웠던 건 비개발자 두 분이 뛸듯이 기뻐하신거에요. 고작 2시간 동안 2-3가지의 variation 을 해보시곤 PR까지 올렸습니다. 인터넷 예제가 아니라 진짜 Clobe의 문제를 풀었습니다. clarify -> plan -> implement -> wrap 4단계로요.

비개발자분들의 접근
- 중요한 컴포넌트에 애니메이션을 추가해서 클릭 전환율 높일 준비
- 네비게이션 바의 위계를 재설정하면서 2-3가지 프로토타입 구현 후 최종 형태 결정
처음엔 CLI 를 어려워하셨지만 익숙해지니까 놀랄 정도로 빠르게 구현하셨어요. 저희는 옆에서 원리를 좀 더 알려드리고, 요구사항을 명확하게 뽑을 수 있게 AskUserQuestion 을 추천해드렸어요.

개발자분들의 접근
- 문의 내용 중에 자동으로 ticket 생성, worktree 전환해서 자동으로 구현
- slack integration hooks
- 방대하게 실제 데이터를 분석해서 새 알고리즘 feature 추천
- ...
개발자분들은 이 자리에서 직접 구현한 것보다 skills 등의 원리를 저희와 깊게 이야기했어요. Claude Code야 원래 사용하고 계셨기 때문에 더 중요한 건 디테일이었던 것 같아요

늦은 시간까지 전혀 지치지 않고 같이 문제를 풀어서 재밌었습니다. 열심히 준비해간 보람이 있었어요. 감사합니다. Jordan Bae 김정원
