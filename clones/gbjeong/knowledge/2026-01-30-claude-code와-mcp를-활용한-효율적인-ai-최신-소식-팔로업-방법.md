AI 최신 소식을 어떻게 팔로업하냐는 질문을 많이 받았습니다. 그냥 많이 읽으려고 노력했는데, 더 잘하려고 내부 프로덕트+MCP를 만들어서 쓰고 있습니다.
1. 1-page digest를 먼저 훑어보기(메일, 슬랙, 웹페이지)
2. Claude Code에서 Contents hub MCP 연결하기
3. Skills: /contents-hub 미리 정의한 content lens로 컨텐츠 클러스터링하고 관심사를 선택
4. 흥미로운 건 원문까지 Claude가 직접 읽어옴
5. 반응, 댓글까지 파악

여기까지 5분도 안 걸립니다.

오늘 이렇게 발견한 것 중 하나가 Every의 AI 프로젝트 매니저였는데요. CLAUDE.md 파일을 Job Description으로 쓴다는 게 재밌더라고요.

AI 소식 팔로업하려고 X, Youtube, RSS를 직접 보지는 않습니다. 관심 가는 정보 빠르게 캐치해서 Claude Code 로 같이 분석합니다.
