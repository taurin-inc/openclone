Claude Code를 코딩에만 사용하지 마세요. 모든 업무의 OS로 사용하세요. 저는 캘린더 skill을 만들어서 사용합니다.
"sync 미팅 30분 뒤로 옮겨줘" 이 한 마디로 캘린더가 바뀝니다.

"내일 호연이랑 1시간 미팅 잡아줘" 하면 바로 생성됩니다.

Google Calendar 플러그인을 만들어서 plugins-for-claude-natives에 추가했어요.

1. 멀티 계정 지원
회사 캘린더, 개인 캘린더를 한 번에 봅니다.
"오늘 일정 알려줘" 하면 둘 다 통합해서 보여줍니다.

2. 병렬 실행
여러 계정을 동시에 조회합니다.
Subagent가 각 계정을 병렬로 가져와서 빠릅니다.

3. 충돌 감지
회사 미팅과 개인 일정이 겹치면 알려줍니다.
더 이상 캘린더 두 개 번갈아 볼 필요 없어요.

제가 가장 유용하게 쓰는 건 주간 회고입니다.
지난 주에 누구와 어떤 미팅이 있었는지 물어보면 정리해줍니다.
Slack skill이랑 같이 쓰면, 미팅 내용과 후속 대화까지 한 번에 파악됩니다.

Claude Code PM Cat Wu가 말했듯이, Claude Code는 개발자만의 도구가 아닙니다.
캘린더 관리처럼 일상적인 업무도 자동화할 수 있어요.

plugins-for-claude-natives에 더 많은 플러그인을 추가할 예정입니다.
