사내 MCP와 에이전트를 만들어서 업무를 자동화한 사례를 공유했습니다. 운영/세일즈 자동화를 위해 클로드코드에게 던진 프롬프트를 공유합니다!

"저희 팀은 직접 구축한 mcp 를 이렇게 사용하고 있어요."

1. 슬랙 스레드로 명령하면 메시지를 읽고 액션하는 에이전트
저희는 회사의 거절 사유를 노티하는데요. 스레드로 에이전트를 불러서 요청만 하면 적절한 mcp tool 을 사용해서 업무를 완료해줍니다. 슬랙봇과 mcp 서버 모두 직접 구현했습니다.

2. mcp 서버를 클로드에 직접 연결해서 쓰기도 합니다.
복잡한 액션은 클로드에서 수행하면 편합니다. 빠르고, 추가비용도 안 드니까요. "*(슬랙 메시지 링크만 붙여넣고) 회사의 피드백을 반영해서 평가기준과 필터 조건을 업데이트해. +추가로 SaaS경험을 필수로 지정해" 7-8번의 mcp 호출을 통해 딱 내가 원하는 값으로 업데이트해줍니다.

3. 5줄의 프롬프트로 MCP 만들기
세일즈에 사용하는 임직원 수 조회 MCP는 간단해서 다섯 줄의 프롬프트로 첫 번째 스코프를 구현할 수 있었습니다.
```
목표: 한국 공공데이터포탈에서 제공하는 국민연금 가입 사업장 내역 API를 mcp server 로 만든다.
스펙: python / uv / pypi 배포
개발 원칙: context7 mcp / 서버 구현체 웹 검색 / 단계마다 테스트하고 통과하면 commit
자료 1. modelcontextprotocol/python-sdk context7 mcp 사용해서 읽기
자료 2. docs/국민연금공단_오픈API활용가이드_국민연금 가입 사업장 내역_v2.0.md 문서 읽기
```

버그 없는 초기버전을 구현할 수 있었던 건 사전조사를 빡세게 해서 맥락을 알려줬기 때문입니다. 물론 조사도 chatgpt를 이용해서 한 거지만요. MCP 서버 구현은 처음이라 공식 문서를 챗지피티와 함께 읽었어요. 이해가 안되는 걸 질문하면서요.

세미나가 끝나고 직접 말씀을 해주시는 분들이 많았습니다. 특히 이번엔 제가 만들었던 걸 공유해서 도움이 됐다고 하셨어요.
