GPT 5.2가 발표됐습니다. 이번 모델은 정확히 Professional work를 타겟합니다. AGI 를 외쳤던 GPT 5, 성격 커스터마이징을 외쳤던 gpt 5.1과 방향성이 다릅니다. Gemini 3.0, Opus 4.5의 Tool for Work이라는 포지션을 노립니다. codex 에서 지금 바로 사용할 수 있어요!

Gemini 3.0 -> Opus 4.5 -> GPT 순서로 또 5.2 모델이 나왔습니다. 아직 저는 ChatGPT 앱에서는 5.2모델이 roll out 되지 않았고요. 대신 "codex -m gpt-5.2" 로 바로 모델을 사용할 수 있었습니다.

introducing 페이지는 자세히 읽어보지 않았어요. 벤치마크야 좋게 만들어서 발표했을테고 이젠 벤치마크가 피로하거든요. 리얼 케이스에서 써봐야 체감이 되는 것 같아요. 

작업 중인 프로젝트에 간단히 써봤는데요. xhigh 모델이 5.1모델보다 더 오랜 시간, 더 많은 토큰을 사용한다고 느꼈어요. 출력 길이도 길어서 왠지 모를 효능감도 높고요. 5.1 모델에서 xhigh 를 발표하면서 token 효율성을 강조했는데요. 그때는 모델을 채찍질해도 많은 토큰을 사용하지 않아서 아쉬웠거든요.

단시간에 내용의 퀄리티를 당장 체감하긴 어렵지만 5.1보다 더 제가 원하는 방향으로 제안을 해주는 것 같아요. 같은 쿼리를 5.1로도 돌려보려고 했지만 5.1에서는 더이상 xhigh 를 지원해주지 않네요. 일부러 비교..하지 못하게 하려고 그런 건 아니겠죠? ㅎㅎ

새로운 모델이 나왔으니 당분간 강제로 써보려고 합니다! 아마 많은 분들이 그러셨던 것처럼 저도 Gemini 3.0을 굉장히 많이 썼어요. 나노 바나나의 역할이 컸고, 사실 GPT 5, 5.1, 5.2 발표 내용의 차이를 분석시킨 것도 Gemini 였구요. 

5.1 모델이 혹평을 꽤 받았다고 들었습니다. Code Red 이후에 나온 이번 모델이 그래서 더 기대가 됩니다!
