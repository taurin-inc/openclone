/add-mcp-server 명령어 하나면 누구나 공공 mcp를 만들 수 있는 오픈소스를 만들었습니다. 클로드코드에 프롬프트를 잘 만들어서 mcp를 만드는 발표를 했습니다. 지피터스 운영진 분들이 제 글을 좋게 보시고 연락을 주셔서 mcp 를 만드는 방법을 이야기해드릴 수 있었습니다

클로드코드가 공공 mcp 서버를 자동으로 만드는 방법입니다. 아래 가이드를 자세한 프롬프트로 작성했고, 매우 잘 작동합니다!
1. API 문서 수집/정리 및 네이밍 자동화
2. 템플릿 복제
3. 코어 로직 작성: api 클라이언트 및 모델
4. mcp 서버 툴 구현
5. 테스트
6. README 문서화
7. pip install 할 수 있게 배포 (mcp를 한 번에 설치할 수 있게)

공공 API 에는 재미있는 데이터들이 많습니다.
국민연금 api 로 임직원 수를, 나라장터 api 로 입찰 데이터를 얻을 수도 있습니다. 독특하게도(?) 역대 대통령 연설문도 공공 api 로 제공하고 있습니다. 나라에서 보증한 출처가 분명한 데이터를 잘 활용하면 더 재미있는 결과가 나올 것 같아요

오픈소스 링크는 댓글로 남겨두겠습니다.
