괜찮은 claude code skill, plugin 관련 프로젝트들 공유드립니다! 좋은 프로젝트를 구경하면 좋은 프로젝트를 만들어내기도 좋은 것 같아요.
1. https://lnkd.in/ghH3uYjy : claude code plugin 을 모아놓은 서비스. skills 가 있는 프로젝트만 따로 필터링할 수도 있음
2. https://lnkd.in/gnEd37iw : 브라우저 오토메이션하는 skill. 스킬 중에 mcp 대비 효율이 단연코 높은 도메인(?)을 뽑자면 브라우저 오토메이션이다. 정확도와 비용은 물론 특히나 속도가 어마어마하게 개선된다. 프로덕션 환경에서 더 안정적인 것은 덤. 어제 browser use 에서 skill 기능을 발표해서 화제가 됐는데, 오픈소스로 써볼 수 있다.
3. https://lnkd.in/gCbhq5Ps : 너무 많은 기능이 들어있어서 내 스타일은 아니지만 1가지가 마음에 듦. thinking-level 0,1,2,3 을 정해두면 항상 think 레벨을 고정할 수 있다. 3=ultrathink
4. https://lnkd.in/gW5CGZUw : BMAD Method v6 for Claude Code. 개인적으로 BMAD에 회의적이었는데 여러 피드백을 받아서 더 자세히 사용해보려고 합니다. claude code 에 괜찮게 통합된 프로젝트가 있어서 공유.
5. https://lnkd.in/gvZafWtG : claude code history analyzer. 자주 사용하는 커맨드는 슬래쉬 커맨드로 만들어주거나, 복잡한 워크플로우는 서브에이전트를 만들게 한다. 재사용 가능한 도메인 지식은 스킬로 만들게 추천. 

+번외: https://lnkd.in/gwe-PU3A 123개의 과학 관련 스킬. Bio, Chem, Medical 등 진짜 과학과 관련된 스킬을 담았다.
