openai 는 광고를 붙였습니다. B2C는 지루해지는데 일을 위한 생태계(편의상 B2B)는 놀랍도록 재밌습니다.
1. Claude Cowork 의 카피가 며칠이면 나옵니다. 
langchain의 openwork, different ai 에서도 openwork라는 이름으로 대체재를 만들었습니다. 
"Ideas are cheap, execution is cheaper" 10년의 커리어 동안 실행은 항상 비쌌는데 이제 실행은 정말 쌉니다. 
https://lnkd.in/gxTnuuUd

2. 고품질의 플러그인 생태계가 만들어지고 있습니다.
oh my opencode 가 전세계 선두 그룹에 있는 플러그인이라고 생각합니다. vercel에서 "react best practice" 스킬을 공개했는데 분명히 oh my opencode 가 영향을 줬을 것이라고 생각합니다. 
+superpowers 도 재밌게 지켜보고 있습니다.
- vercel agent skills: https://lnkd.in/gqvuqaMY
- superpowers plugin: https://lnkd.in/gDiQaEQx

3. 서드파티 앱이 LLM API 에 의존하는게 아니라 Claude Code 생태계에 의존합니다. 
Cowork는 물론, opencode 같은 앱들이 공식/비공식적으로 Claude Code의 생태계에 아주 강하게 의존합니다. 사실상 mcp, skills 를 만들어낸 Claude 를 codex, cursor 가 따라오고 있습니다. 이미 SF에서 개발자 설문을 해보면 Claude Code 사용량이 Cursor를 넘었습니다.

벌써 작년에 배웠던 지식 중에 unlearn해야할 게 보입니다. 2026년이 너무 기대됩니다!
