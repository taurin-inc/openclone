딱 한 줄이면 클로드 코드가 gpt를 부를 수 있어요. chatgpt 구독자는 codex를 api 비용 없이 쓸 수 있게 정책이 업데이트됐거든요.  
`claude mcp add codex --scope user codex mcp`

gpt5에서 코딩 성능이 많이 좋아졌으니 꼭 연동해서 필요할 때 쓰시는 걸 추천드립니다.
저는 계획 단계에서 opus + gpt5를 같이 써보고 있습니다. 토큰을 더 많이 쓸 수 있으니까 더 좋은 계획을 세울 수 있는 것 같아요. 

하지만 codex 호출이 워낙 장시간 돌아가게 설계돼있어서 조그마한 작업에도 오래 걸린다는 느낌을 지울 수가 없네요. 신중하게 사용하는게 좋을 것 같습니다. 

+놀랍게도 codex cli 에서 직접 제공한 기능입니다. 클로드 코드에 붙여 쓸 거라고 예상했을텐데 정면 승부를 하는 걸까요? 아니면 이렇게라도 개발자들에게 gpt5를 사용하는 경험을 제공하려는 걸까요. 어느쪽이든 소비자의 입장에서는 신이 나네요!

claude code가 메인이더라도 codex cli 까지 붙여서 토큰 알차게 사용해보시길 바랍니다!
