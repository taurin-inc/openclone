요새 시간 가는 줄 모르고 사무실에 있어요. 좋은 동료가 생겼기 때문인데요. AI로 개발하기, 오픈소스, 사람 만나서 배우기, 스타트업까지 주제가 아주 잘 맞아요. 하지만 진짜 좋은 건 믿을 수 있는 분들이라는 거에요.

[사람들]
Jake Park 건태님은 궁금한 걸 못 참아서 싱크를 계속 맞춰요. 너무 좋아요.
HoYeon Lee 호연님은 온도가 높아요. ~~니까 더 잘 할 수 있다! 라고 자주 해요.
Changhoi Kim 창회님은 왜? 라는 질문을 많이 해요. 근데 하고나면 활짝 웃어요.

[공간]
다 같이 카페에서 일했던 날, Google Startup Campus 에 있었던 적도 있어요. Popow 팀에서 배려해줘서 오피스 공간에 살짝 sneak 하기도 했어요. 그러다 고위드 팀에서 배려해주셔서 지금은 호림아트센터 14층에 둥지를 틀었습니다!

[오픈소스]
건태님은 Code Squad라는 vs code plugin 을 만들었어요. 저희는 Cursor를 구독하지 않고 건태님의 플러그인을 써요. 그러면 Claude Code, Codex, Gemini CLI 구독만 해도 커서의 UI를 쓸 수 있어요. 더 좋은 것들도 있고요. 덕분에 다시 IDE 맛깔나게 씁니다.

호연님은 ctx 라는 개발할 때 컨텍스트를 잃지 않는 프레임워크를 만들었어요. 또 어떤 페이지든 컨텐츠가 새로 올라오는 걸 구독하는 프로젝트도 했는데, 이건 서비스로 만들기 위해서 private 으로 바꾼 상태에요. 

저는 agent council 이라는 claude code가 codex, gemini cli 를 불러다 쓰는 skill을 만들었어요. 간단한 프로젝트인데 누군가의 PR을 받고 기능이 추가되는게 너무 재밌어요. 감사할 따름입니다.

창회님은 바쁜 일을 정리했고 이제 시작할 준비를 하고 있고요.

[더 많은 사람들]
어제는 저희가 '와 이 사람 AI 진짜 잘 쓴다' 라고 느낀 분들을 저녁에 초대했어요. 그리고 사람한테 배우는 게 진짜 지식이라고 생각이 더 확고해졌어요. 그리고 배워서 좋은 것 뿐만 아니라 그냥 사람들이 좋더라구요. 7시에 시작해서 10시까지 시간 가는 줄 몰랐어요. 수다 떨기, 자주 있었으면 좋겠다, 최고였다 라는 간증을! 저도 배운게 너무 많은데 이건 찬찬히 풀어보겠습니다.

[커뮤니티 빌드]
사람들이 모여서 진짜 지식이 생기고, 그걸 바깥에 공유하는 것. 그리고 이걸 토대로 AI 기반으로 생산성을 파괴적으로 올리는 생태계를 서포트하고 싶어요. 기존에는 대부분 개발 스택을 기반으로 커뮤니티를 만들었어요. 대부분의 사람은 개발 언어 하나에 종속되거든요. 

그런데 어제는 언어 얘기는 하나도 안 했어요. 프레임워크 얘기도 거의 안 나왔어요. 클로드 코드, 지식 관리, Qwen, 24시간 자동으로 AI 돌리는 법, 자동으로 회사 돌리는 법 이런 얘기들이었어요. AI 로 일을 하면 접근법이 완전히 달라지는 것 같다고 느껴요. 이런 생태계에 기여하고 싶고, 좋은 기여를 할 수 있을 것 같아서 기쁩니다!
