1월 1일까지 Claude Code, Codex 사용량이 2배로 늘어났습니다. 경험치 2배 이벤트를 놓치지 마세요! 그리고 저는 요새 AskUserQuestion이라는 Tool을 Claude Code 에서 정말 자주 사용합니다.

사용량 2배 이벤트, Claude가 먼저 시작하고 Codex가 따라오네요. 연말에 우리나라도 그렇지만 미국도 회사 업무가 느슨해지는 경우가 많아서 이런 이벤트를 잘 이용해서 못해왔던 프로젝트를 하면 좋을 것 같습니다.

2개 다 훌륭한 툴인데, AskUserQuestion 툴 때문에 Claude Code 를 버릴 수가 없어요. Plan mode 를 사용하면 자주 나오는 객관식 질문 툴인데, 명시적으로 AskUserQuestion 해서 모호한 내 요구사항을 구체적으로 바꿔달라고 하면 여러 번 질문하도록 가이드할 수 있습니다.

그냥 질문을 해달라고 하면 주관식으로 물어보는데, 이 툴을 명시하면 Claude Code 의 객관식 선택 뷰를 사용할 수 있어서 매우 편리합니다. 이 방식으로 연말에 Claude Code와 친해져보세요!
