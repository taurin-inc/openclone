Databricks가 Neon을 약 1조 4,000억원에 인수했습니다.

LLM 기반 애플리케이션을 구축하고 계시다면, Neon을 적극 추천드립니다. 저희 팀은 Neon 위에 Hybrid Search를 구현해 사용 중인데, 저렴하면서도 Postgres 기반 SQL 스키마를 그대로 활용할 수 있어 매우 만족하고 있습니다.
기존엔 Pinecone으로 Vector & BM25 검색을 구현했지만, 메타데이터가 많으니 비용이 급증했는데요. Neon은 Paradedb(pg-search)를 바로 사용할 수 있어 세팅이 훨씬 간단합니다.

LLM 의 등장 타이밍과 맞물려서 데이터베이스 생성 수요가 급증했다는 소식을 들었는데, Neon은 이런 흐름을 정확히 읽고 빠르게 움직인 것 같아요.
2021년 창업 → 2025년 Databricks 인수라니, 인상적인 여정입니다. 그냥 제가 좋아하는 서비스라 기분이 좋네요 ㅎㅎ
