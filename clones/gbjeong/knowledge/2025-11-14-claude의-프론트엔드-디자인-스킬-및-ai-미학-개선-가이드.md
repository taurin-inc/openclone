LLM이 또 흰 배경에 보라색 그라데이션 웹페이지를 만들어줬나요? 이틀 전에 추가된 Claude frontend-design skill 을 써을 써야합니다.. +Anthropic 에서 제공하는 기본 스킬도 다운로드 받는 plugin까지

LLM한테 랜딩 페이지를 만들어달라고 하면 흰 배경 위의 보라색 그라데이션을 만듭니다. 왜냐면 이런 디자인이 실제로 많기 때문이죠! 하지만.. 못생겼습니다.

Anthropic 에서 가장 범용적인 프롬프트로 깎아서 400토큰짜리 스킬로 만들어줬습니다.
```
<frontend_aesthetics>
당신은 일반적이고 "분포상" 출력으로 수렴하는 경향이 있습니다. 프론트엔드 디자인에서 이것은 사용자가 "AI 쓰레기" 미학이라고 부르는 것을 만듭니다. (생략...)

다음에 집중하세요:
- 타이포그래피: arial, inter 피하기
- 색상 & 테마: 일관성을 위해 CSS 변수를 사용. 악센트가 있는 주요 색상 사용, 팔레트 피하기
- 모션: 효과와 마이크로 인터랙션에 애니메이션 사용. React의 경우 가능할 때 Motion 라이브러리를 사용
- 배경: 단색으로 기본 설정하는 대신 분위기와 깊이, CSS 그라데이션을 레이어화하고, 기하학적 패턴을 사용하거나, 전체 미학과 일치하는 맥락적 효과를 추가하세요.

일반적인 AI 생성 미학을 피하세요:
- 과도하게 사용되는 폰트 계열(Inter, Roboto, Arial, 시스템 폰트)
- 진부한 색상 구성(특히 흰색 배경 위의 보라색 그라데이션)
- 예측 가능한 레이아웃과 컴포넌트 패턴
- 맥락별 특성이 부족한 틀에 박힌 디자인

창의적으로 해석하고 맥락에 맞게 진정으로 디자인된 느낌을 주는 예상치 못한 선택을 하세요! (생략...)
</frontend_aesthetics>
```
