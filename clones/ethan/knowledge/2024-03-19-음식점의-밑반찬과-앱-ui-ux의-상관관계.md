<사Up, 스타트Up, Never Give Up>

1. 처음에 나오는 반찬만 먹어보면 이 집이 맛집인지 판별이 가능하다고 흔히들 말한다

2. 맛집이라는 것 자체가 사실 극히 주관적인 것이나, 경험상 대체적으로는 맞는 말 같다

3. 복국집 가서 복국은 안 먹고 깍두기만 절단 내는 지인도 있다

4. 여튼

5. 반찬이라 하는 것은 사실 극동 아시아 특유의 문화인데 우리나라처럼 묻지도 따지지도 않고 웬만하면 무한리필인 국가는 아직 경험해보지 못했다

6. 그래서 사실 한국 음식점의 반찬은 경제적 논리에 의하면 맛이 없어야 한다

7. 그래야 사람들이 더 달라 안 하고, 원가 지출이 덜 될 테다

8. 그런데도 기어이 밑반찬을 맛있게 만들어서 리필을 촉진시키는 곳들은 어리석은 걸까

9. 주메뉴와의 궁합

10. 명동칼국수 하면 (예전) 김치, 쵸큼 고급진 중국집 하면 짜사이, 제주 돼지고기집 하면 멜젓

11. 돈 받는 음식은 누구나.. 는 아니고 대부분 잘 만든다

12. 돈 안 받는 조연은 누구나 신경쓰지는 않는다

13. 조연으로 주연을 압도하겠냐마는 조연이 약하면 주연도 빛 바래기 쉽다

14. 이 역할을 앱에서는 ui ux가 한다

15. 코어 콘텐츠가 좋아도 ui ux가 약하면 한 번 망설이게 된다

16. 쿠팡플레이 ui ux가 부쩍 좋아졌다

17. 유튜브 ui ux는 솔직히 아쉽다 화면잠금도 없고...

18. 쿠클에서 Retention 분석을 할 때 ui ux를 많이 분석했었다

19. 식당의 주력 메뉴를 바꿀 슈는 없지만 반찬, 인테리어, 간판만 바꿔도 일단 손님은 늘어난다

20. 그런데 새조개는 참 비싸다 

#다음중문맥에어울리지않는말은 #uiux
