AI 에이전트가 정말 사람을 대체할지, Scale AI가 실험했습니다

실제 프리랜서 플랫폼에서 거래된 240개 프로젝트를 모았습니다. 건축 설계, 게임 개발, 3D 모델링, 음악 작곡, 데이터 분석 등 평균 28.9시간이 걸리는 일들이며 총 가치는 $143,991 입니다. 그리고 이를 해결하기 위해 6개의 최첨단 AI 에이전트를 투입했습니다. GPT-5, Claude Sonnet 4.5, Grok 4, Gemini 2.5 Pro. 

결과는 1등의 성공률도 단 2.5%였습니다. 40개 일감 중 1개만 제대로 완료했습니다.

1. 진짜 시장에서 진짜 일을 가져왔습니다

기존 AI 벤치마크는 제한된 작업입니다. 코딩 테스트, 수학 문제, 웹 검색 등 고립된 능력을 측정합니다. 하지만 실제 프리랜서는 클라이언트의 모호한 요구사항을 이해하고, 여러 파일을 다루고, 스스로 품질을 확인하고, 디테일을 조정합니다. 연구팀은 이 모든 복잡성을 담고 싶었습니다. 실제 프리랜서 358명을 모집했습니다. Upwork에서 평균 2,341시간 작업, 89개 프로젝트 완료, $23,364 수익을 올린 프로들입니다. 그들에게 개당 $15~200을 지불하고 실제로 클라이언트에게 돈 받고 완료한 작업 샘플을 구매했습니다.

클라이언트가 제공한 요청서, 필요한 모든 입력 파일, 프로가 납품한 최종 결과물, 550개가 모였습니다. 그 중에서, 지시사항이 완전한가? 외부 정보 없이 완수 가능한가? 민감한 정보는 제거했는가? 등의 여러 차례 검토를 거쳐 240개로 압축했습니다. 각 프로젝트는 완전히 독립적으로 완료 가능하도록 표준화했습니다. 최종 데이터셋으로, 23개 직업 카테고리, 수십 가지 파일 형식, 평균 완료 시간 28.9시간의 진짜 프리랜서 시장의 복잡성을 담은 벤치마크가 준비되었습니다.

2. 합리적인 클라이언트가 돈을 낼까로 평가했습니다

AI에게 각 프로젝트의 요청서와 입력 파일을 줬습니다. 프로젝트당 API 호출 예산은 최대 $30로 실제 프리랜서의 시간당 비용을 고려하였습니다. AI가 결과물을 제출하면, 전문 평가자 3명이 독립적으로 평가합니다. 인간 프로의 결과물과 비교하면서 묻습니다. "합리적인 클라이언트가 이 결과물에 돈을 낼 것인가?" 평가자는 클라이언트의 요청서를 읽고, 인간 결과물을 확인하고, AI 결과물을 파일 하나하나 열어봅니다. CAD 설계도를 돌려보고, 비디오를 재생하고, 3D 모델을 확대하고, 웹 게임을 실행합니다. 평균 11.4분이 걸립니다. 3명의 다수결로 결정합니다.

3. 결과는 1등도 단 2.5%만 성공했습니다
- Manus: 2.5% - $1,720
- Grok 4: 2.1% - $858
- Claude Sonnet 4.5: 2.1% - $1,280
- GPT-5: 1.7% - $1,180
- ChatGPT Agent: 1.3% - $520
- Gemini 2.5 Pro: 0.8% - $210

AI는 어디서 잘못했을까요? 연구팀은 수백 건의 실패 사례를 분석했습니다. 패턴은 있었습니다.
- 파일이 깨지거나 비어있음 (17.6%): 기술적 문제. 잘못된 형식으로 제출.
- 결과물이 불완전함 (35.7%): 일부만 완성. 비디오가 중간에 끊김. 필요한 파일 누락.
- 품질이 프로 수준 미달 (45.6%): 완성은 했지만 클라이언트가 받아들일 수 없는 품질.
- 파일 간 불일치 (14.8%): 3D 모델의 각 뷰마다 제품 외관이 다름. 스타일 일관성 없음.

구체적 사례를 보면, 기존 파일을 정밀하게 수정하기, 복잡한 다단계 지시사항 따르기, 여러 요구사항 동시에 충족하기, 결과물을 검증하고 수정하기와 같은 작업들이 되지 않았습니다.
- 8분 비디오 요청 → 8초짜리 제출
- 건축 설계도 → 3D 뷰마다 집 외관이 다름
- 웹 게임 → 작동하지만 기본 도형만 쓴 어린이 그림 수준
- 음성 → 로봇처럼 부자연스러움
- 실측 스케치 → AI가 만든 도면이 치수 안 맞음

4. 연구팀의 결론: AI의 생성 능력은 높지만, 디테일 지향적 전문가로서의 신뢰성은 낮다

CAIS 디렉터 Dan Hendrycks는 근본 원인을 지적합니다. "장기 기억이 안 됩니다. 경험으로부터 지속 학습이 불가능합니다. 인간처럼 직무 중 기술을 습득하지 못합니다." 인간 프리랜서는 납품 전에 자기 작업을 검토합니다. 클라이언트 입장에서 봅니다. 빠진 것은 없는지, 품질은 괜찮은지, 요구사항을 다 충족했는지. 이 "자기 검증" 능력이 프로와 아마추어를 가릅니다. AI는 이 능력이 없습니다. 8분 비디오를 만들라고 했는데 8초짜리를 만들고도 그게 틀렸다는 걸 모릅니다. 3D 모델의 각 뷰가 일관성이 없어도 확인하지 못합니다.

"AI와 일자리"에 대한 논쟁은 수년간 있었습니다. 하지만 대부분 추측 중심이었습니다. "AI가 단순 반복 작업을 대체할 것이다." "창의적 작업은 안전하다." "5년 안에 프리랜서의 30%가 사라진다." 하지만, 이제 데이터가 있습니다. 이번 실험은 실제 경제적 가치를 기준으로 AI의 자동화 능력을 측정했습니다. 결과는 2.5% 였습니다. 6개월 후, 1년 후, 이 숫자가 어떻게 변할까요?

AI가 사람을 완전하게 대체하기는 아직 시간이 걸릴 것 같습니다. 하지만 이것이 "AI가 쓸모없다"는 뜻은 아니라고 생각합니다. 숙련된 사람의 손에서 AI는 매우 강력한 도구입니다. 당분간 노동시장의 변화는 대체가 아닌 증강에서 일어날 것으로 보입니다.
