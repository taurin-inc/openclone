Claude for Financial Services가 출시되었습니다

확실히 OpenAI와는 다른 길을 가는 것 같습니다. OpenAI는 범용성에 집중합니다. ChatGPT Enterprise를 만들고, Google Drive와 같은 범용 커넥터를 제공해서 기호에 맞게 사용 가능합니다. 회사마다 필요한 것을 연결하고, 원하는 기능을 만들고, 프롬프트를 잘 짜서 활용할 수 있습니다.

Anthropic은 조금은 다르게 가는 것 같습니다. 지난번에 스킬 기능을 발표했을 때만 해도 "사용자가 직접 만들어 쓰세요”인 것 같았는데, 이번에는 아예 스킬을 만들어 주었습니다. 대략 6개를 발표하였습니다.

1. 파이낸스 스킬을 직접 만들 필요가 없습니다

DCF 모델, 동종업계 비교 분석, 실사 데이터팩, 회사 소개서, 어닝 분석 등 투자 검토할 때 필요한 것들을 만들어뒀습니다. 프롬프트를 자세하게 칠 필요도 없습니다. "Tesla DCF"라고만 치면 잉여현금흐름 전망부터 WACC 계산, 시나리오 분석까지 나옵니다. 그냥 스킬을 활용하면 복잡한 분석이 됩니다. 직접 만들 필요가 없습니다.

2. 데이터 연동까지 해뒀습니다

S&P Capital IQ, FactSet, PitchBook, Moody's 등 다양한 금융 데이터 제공업체와 이미 연결되어 있습니다. 사용자가 Bloomberg에서 데이터 다운받아서 업로드할 필요가 없습니다. 이미 되어 있습니다. Excel 애드인도 베타로 나왔습니다. Excel 사이드바에서 Claude를 바로 쓸 수 있습니다. 복사-붙여넣기 없이 Excel 안에서 모든 게 끝납니다.

3. 투자 검토 방식이 바뀔 것 같습니다

이제 많은 투자자, 심지어 벤처캐피탈까지도 일하는 형태가 바뀔 것 같습니다. 투자 검토의 본질은 자료 만드는 것이 아닙니다. 왜 이 투자를 해야 하는지에 대한 Thesis를 짜기 위해 깊은 고민과 통찰이 필요합니다. 이 회사가 시장을 어떻게 바꿀 것인지, 경쟁자들과 무엇이 다른지, 창업자가 이걸 해낼 수 있는 사람인지, 이런 질문들에 고민하는 것이 본질에 가깝다고 생각합니다.

하지만 많은 심사역들은 절차와 보고서 정리에 시간을 쓰고 있습니다. 산업 리포트 읽고 요약하고, 동종업계 비교하고, 재무 모델 만들고, 실사 자료 정리하고, 그러다 보니 투자까지의 기간도 보통 한 달 이상 걸립니다. Claude for Financial Services는 이런 작업 시간을 몇 분으로 줄입니다. 실사 자료 정리에 이틀 걸리던 게 몇 분이 됩니다. DCF 모델 만드는 데 반나절 걸리던 게 즉시 나옵니다.

그러면 심사역은 무엇을 할까요? 생각할 시간이 생깁니다. AI를 잘 써서 불필요한 작업을 줄이고, 생각하는 시간을 더 늘리면서, 의사결정을 빠르게 하는 VC와 그렇지 않은 VC의 차이는 있을 것 같습니다. 같은 딜을 검토하는데 한 곳은 2주, 한 곳은 6주가 걸린다면요? 창업자 입장에서는 누구를 선택할까요?

물론 앞으로 보안 등의 문제로 인해 빠르게 도입되기는 어렵겠습니다. 금융 데이터는 민감하고, 규제도 많습니다. "입력과 출력 데이터를 모델 학습에 사용하지 않는다"고 명시하고 있지만, 내부 데이터를 외부 AI에 넘기는 것에 대한 거부감은 여전히 있을 겁니다.

하지만 변화는 시작되었습니다. AI로 할 수 있는 것들은 점점 더 많아지고 있습니다. 그리고 사용하기도 점점 더 쉬워지고 있습니다. 도태되지 않기 위해 이러한 변화에 빠르게 적응하는 것이 점점 더 중요해지고 있습니다.
