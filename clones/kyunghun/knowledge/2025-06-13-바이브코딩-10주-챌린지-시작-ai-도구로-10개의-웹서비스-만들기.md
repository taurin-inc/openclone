🚀 바이브코딩 10주 챌린지 시작합니다!

AI 도구들이 이렇게 발전했는데, 실제로 얼마나 빠르게 서비스를 만들 수 있을까요?
매일 새로운 AI 개발 도구들이 쏟아지고 있는 지금, 직접 써보면서 배우는 게 정말 중요하다고 생각했습니다.
그래서 시작하는 바이브코딩 10주 챌린지입니다.

📋 챌린지 룰
- 목표: 10주 동안 매주 1개씩, 총 10개의 실제 사용할 수 있는 웹서비스 만들기
- 핵심: Cursor 같은 AI 도구들을 최대한 활용해서 혼자서도 빠르게 완성도 있는 서비스 만들어보기

🎯 만들어볼 프로덕트 선정 기준
GPT와 함께 브레인스토밍하면서 3가지 기준으로 골랐습니다.
1. 난이도가 적절한가? - 너무 어렵지 않으면서 매주 완성 가능한 수준
2. 기술적 학습 효과가 있는가? - API 연동, 이미지 처리, 데이터베이스 등 단계별로 새로운 기술 경험
3. 실제로 써보고 싶은가? - 만들고 나서 나 스스로도 계속 사용해볼 만한 서비스

🛠️ 주차별 프로덕트
(W1) 뉴스 밈 카드 제조기 - 뉴스 링크 넣으면 SNS용 재미있는 카드 자동 생성
(W2) 오늘의 동기부여 - 랜덤 명언 + 예쁜 카드 디자인
(W3) AI 운세보기 - 화투 뽑으면 GPT가 오늘 운세 해석해주기
(W4) 1분 피치덱 생성기 - 사업 아이디어 한 줄 입력하면 투자자용 랜딩페이지 자동 완성
(W5) 우리동네 나들이 추천 - 위치 기반으로 아이와 갈 만한 곳 TOP3 추천
(W6) 아이디어 평가봇 - 사업 아이디어 입력하면 AI가 5점 만점으로 현실적 평가
(W7) 기분별 플레이리스트 - 오늘 기분 선택하면 Spotify에서 딱 맞는 음악 추천
(W8) 꿈 해몽 + 그림 - 꿈 이야기하면 의미 해석 + 몽환적인 그림까지
(W9) 음식 사진 영양 분석 - 사진 찍으면 칼로리, 영양소 자동 계산
(W10) 거북목 측정기 - 웹캠으로 자세 체크하고 교정 운동 가이드

🔧 사용할 도구들
- AI 개발 도구: Cursor, GitHub Copilot, v0, Lovable 등
- API: OpenAI GPT-4, DALL-E 3, Stable Diffusion, Google Vision API 등
- 프론트엔드: Next.js, Tailwind CSS, shadcn/ui 등
- 백엔드: Supabase (Database + Auth + Storage)
- 배포: Vercel, Replit 등

매주 진행상황과 시행착오 과정, 그리고 AI 도구 사용 꿀팁들을 계속 공유하겠습니다!
