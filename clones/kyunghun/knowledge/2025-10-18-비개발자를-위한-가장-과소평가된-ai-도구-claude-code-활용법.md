Claude Code는 비개발자를 위한 가장 과소평가된 AI 도구입니다

Lenny가 500명에게 물었습니다. “Claude Code를 어떻게 쓰시나요?” 답변 중 절반 이상이 코드와 무관했습니다. 회의 녹취록에서 자신이 갈등을 회피한 순간 찾기, 세금 신고용 영수증 파일 자동 정리, 아이를 위한 미끄럼틀 설계하기, 산책하며 녹음한 메모를 블로그 포스트로 변환하기.

1. 이름은 “Claude Code”지만, 실제로는 “Claude Local”

컴퓨터에서 직접 실행되는 AI 에이전트입니다. 클라우드 챗봇의 한계를 벗어나 대용량 파일을 다루고, 제한 없이 오래 실행되며, 로컬 파일과 폴더에 직접 접근합니다.

2.  “코드”라는 이름 때문에 비개발자에게 생기는 두려움

클라우드 챗봇은 우리가 복사/붙여넣기한 정보만 알 수 있습니다. 반면 Claude Code는 폴더 구조를 탐색하고, 파일을 읽고, 시스템 상태를 확인하고, 직접 작업을 실행합니다. 터미널 명령어를 몰라도 됩니다. 그냥 일상적인 언어로 말하면 됩니다.

Martin Merschroth는 세금 신고용 영수증 정리에 씁니다. 폴더 속 난잡한 파일들을 읽고, ‘YYYY-MM-DD 공급업체 - 청구서 - 품목.pdf’로 이름을 바꾸고, 적절한 폴더로 옮깁니다. 코드 한 줄 없습니다. 그냥 “내 영수증 좀 정리해줘”라고 말하면 됩니다.

3. “일머리 좋은 어시스턴트”가 바로 이런 것

Justin Dielmann은 홈 디렉토리에서 Claude Code를 실행합니다. 개인 정리 어시스턴트로 씁니다.
- “중복 파일 찾아서 어떤 걸 남길지 결정 도와줘”
- “이 다운로드들을 적절한 폴더로 정리해줘”
- “내 폴더 구조 검토하고 개선안 제시해줘”

여기서 일어나는 일은 명령어 실행뿐이 아닙니다. 맥락을 이해하고 판단하는 것입니다. 어떤 파일이 중요한지, 어디에 두는 게 합리적인지, 무엇을 지워도 되는지 파악합니다.

4. 스스로 계획하는 플랜 모드

Justin Bleuel은 Claude Code로 채용 관련 자료 일체를 만들었습니다. 직무 설명서, 채용 계획, 면접 계획, 평가 기준까지. 내부 문서들을 폴더에 넣고, 다른 회사의 유사 직무 설명서들을 추가하고, 플랜 모드에서 “이 새 역할에 대한 같은 종류의 자료들을 만들어줘”라고 요청했습니다.

플랜 모드는 Claude Code가 여러 단계의 작업을 스스로 계획하고 실행하게 합니다. 한 번의 질문으로 끝나지 않습니다. 필요한 파일들을 찾고, 읽고, 분석하고, 새로운 것을 생성하는 전체 워크플로우를 설계합니다.

5. 두려움을 내려 놓기

저도 어제 바로 VS Code에 Claude Code를 설치했습니다. 링크드인 글 작성을 위해 개요를 함께 다듬고, 리서치를 진행하고, 각 섹션을 리뷰했습니다. 터미널 명령어 몰라도, 코딩 몰라도, 그냥 일상 언어로 말하면 됩니다.

AI 시대에 가장 중요한 것은 역시 두려움을 내려 놓는 것 입니다.
