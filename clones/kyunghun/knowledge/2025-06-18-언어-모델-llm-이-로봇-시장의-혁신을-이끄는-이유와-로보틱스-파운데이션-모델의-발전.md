언어 모델인 LLM이 왜 로봇 시장의 혁신을 이끌고 있는가?

로봇을 움직이는 AI에도 LLM 기술이 필요할까요? 언어모델이 추론 과정에서 도움이 될 수 있지만 로봇의 경우는, 각 관절과 부품의 움직임을 제어해야 하는 것이 중요하기 때문에, 언뜻 보기에는 서로 큰 관련이 없어 보일 수 있습니다.

하지만 최근 미국을 중심으로 로보틱스 파운데이션 모델이 급성장하고 있는 배경을 살펴보면, LLM이 촉발한 영향이 지대한 것으로 보입니다.

1. 가장 핵심은 로봇 데이터의 토큰화
2018년 이후 LLM 열풍을 일으킨 GPT 등은 수십억 개의 텍스트 데이터로 학습되어 놀라운 일반화 능력을 보여주었습니다. 특정 도메인에 얽매이지 않는 추론 능력과 문제 해결 능력을 갖춘 이들의 성공이 로봇 분야에도 적용될 수 있다는 가능성을 제시했습니다.

핵심은 행동의 토큰화였습니다. 과거 로봇 제어는 연속적인 신호와 전용 네트워크로 처리되어 언어 모델의 지식을 활용하기 어려웠습니다. 하지만 로봇이 관측하는 이미지와 관절 제어 명령을 모두 시퀀스 토큰으로 변환하여 자연어를 처리하듯 동일한 Transformer 모델이 처리하도록 하는 전략이 등장했습니다.

이를 통해 로봇의 동작이 언어처럼 추상화된 토큰 시퀀스로 다뤄지게 되었고, 고차원 시각 정보부터 저차원 액션까지 단일 네트워크에서 학습이 가능해졌습니다. 기존 VLM(Vision-Language Model)이 시각과 언어만 통합했다면, 이제는 VLA(Vision-Language-Action)로 진화하여 행동까지 하나의 통합된 표현 공간에서 처리할 수 있게 되었습니다.

2. 학습 효율성의 폭발적 향상
토큰화의 첫 번째 혁신은 학습 효율성이었습니다. 기존에는 각 구성 요소가 개별 학습되어 모듈 간 정보 손실이 불가피했지만, 모든 데이터가 동일한 토큰 형태로 변환되면서 단일 Transformer 모델이 end-to-end 최적화를 수행할 수 있게 되었습니다.

Physical Intelligence에 따르면, 기존에 400번의 시연이 필요했던 작업을 20-40번만으로 동등한 성능을 달성할 수 있게 되었습니다. 더 나아가 제로샷 능력이 출현했습니다. DeepMind의 RT-2는 "쓰레기 버리기"를 직접 학습하지 않고도 수행할 수 있었고, 새로운 상황에서 성공률이 32%에서 62%로 2배 향상되었습니다.

3. 인터넷 지식과의 혁신적 연결
두 번째 혁신은 더욱 근본적이었습니다. 로봇 데이터가 토큰화되면서 인터넷의 텍스트나 이미지 데이터와 동일한 형태가 되어, 웹에서 학습한 모든 지식을 로봇 제어에 직접 활용할 수 있게 되었습니다.

기존에는 "사과는 빨간 둥근 과일"이라는 웹 지식을 로봇 제어 명령과 연결하기 위해 복잡한 매핑이 필요했습니다. 하지만 토큰화 이후에는 웹 이미지 토큰이 물체 개념 토큰으로, 행동 토큰으로, 실제 모터 제어 토큰으로 자연스럽게 변환되는 통합 파이프라인이 구현되었습니다. RT-2는 웹에서 학습한 "쓰레기는 휴지통에 버린다"는 상식을 바탕으로 처음 보는 쓰레기도 휴지통에 버리는 행동을 생성할 수 있었다고 합니다.

4. LLM 활용 방식의 진화
로봇에 LLM을 적용하는 방식은 점진적으로 발전해왔습니다.
- 토큰화 시작(2022년): RT-1이 로봇 행동을 11개 이산적 파라미터로 토큰화하여 출력
- VLM 통합(2023년 초): PaLM-E가 시각 정보를 LLM에 통합하여 VLA 모델의 기반을 마련
- VLA 구현(2023년 중반): RT-2가 로봇 액션을 텍스트 토큰 시퀀스로 취급하여 복잡한 추론이 가능한 로봇 제어를 실현
- VLA 고도화(2023년 후반-2024년): RT-X가 22개 로봇의 100만 궤적 데이터로 크로스-임바디먼트 학습을 구현, π0에서 연속적 제어 신호를 통합하여 빨래 개기 같은 복잡한 가사 동작을 수행

5. 앞으로의 과제
토큰화 기반 로봇 모델은 혁신적이지만 실시간 제어 측면에서 한계가 있습니다. 대규모 Transformer 모델의 추론 속도가 밀리초 단위 반응이 필요한 로봇의 요구를 완전히 만족시키지 못하기 때문입니다. 경량화 모델들이 해결책을 제시하고 있지만, 성능과 효율성 사이의 트레이드오프는 여전한 과제입니다.

그럼에도 토큰화가 열어놓은 가능성은 명확합니다. 로봇이 단순한 자동화 도구에서 지능적인 협력자로 진화할 기반이 마련되었고, 로보틱스 파운데이션 모델의 급속한 성장을 견인하고 있다고 생각합니다.
