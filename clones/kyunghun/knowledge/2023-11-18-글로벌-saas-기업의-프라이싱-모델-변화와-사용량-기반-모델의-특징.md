Global SaaS 기업들의 프라이싱 모델 변화

- 1990년대: 업프론트 모델 (세일즈 기반 성장)
- 2000년대: 리커링 모델 (마케팅 기반 성장)
- 최근: 사용량 모델 (프로덕트 기반 성장)

사용량 기반 모델의 특징

1. 사용한 만큼만 내는 것이기에 고객입장에서 합리적
2. 작은 규모 고객도 쉽게 가입하여 유저수 크게 확대
3. 고객이 성장하며 지불 금액은 자연스럽게 확대
4. 큰 규모 고객의 매출 비중이 확대 (상위 1%가 40%)
5. 단, 프로덕트 정말 잘 만들어야됨 (진정한 PLG)
6. 프로덕트 사용량과 고객 성장이 직결되어야됨
7. 실제로 사용량 기반 모델은 SaaS 평균보다 매출 성장률, NDR, PSR 등의 전반적 지표 더 좋음
