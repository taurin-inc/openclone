이제 모든 인터페이스는 정말로 에이전트를 통하게 될 것만 같습니다

설마 그렇게 될까? 하는 막연한 생각은 들었습니다. 뭔가 더 불편하지 않을까? 쇼핑을 생각해보면 제가 명확하게 살 것이 있는 경우는 확실히 에이전트가 편할 것 같기는 합니다. 하지만 쇼핑이 꼭 그렇지만은 않습니다. 탐색이라는 과정이 있습니다. 이번 겨울에 코트 뭐사지? 라는 생각에 네이버도 가보고 무신사도 가봅니다. 스크롤하다가 발견하는 순간들이 존재합니다.

커머스만 그럴까요? 어플리케이션도 생각해보면 세일즈포스에는 엄청 기능이 복잡하고 UI도 버튼도 정말 많습니다. 대시보드, 리포트, 워크플로우 등등 이게 그렇게 간단하게 ChatGPT 화면으로 들어갈 수 있을까요?

하지만 조금 더 찬찬히 생각해보니 그럴수도 있겠다 하는 생각이 들었습니다.

1. VIP는 직접 돌아다니지 않습니다

백화점 VIP 라운지를 가보지는 않았기에 상상을 해보면, VIP는 가만히 앉아 있습니다. 쇼퍼가 옷을 골라옵니다. 내 취향을 알고, 체형을 알고, 예산을 알고, 큐레이션해서 가져옵니다. VIP는 탐색을 안 할까요? 아닙니다. 쇼퍼가 대신 탐색해줍니다.

일반 고객은 층마다 돌아다니며 직접 봅니다. 시간은 오래 걸리지만 우연한 발견도 있기는 합니다. VIP는 시간을 절약하고, 전문가의 큐레이션을 받습니다. 탐색은 여전히 일어납니다. 다만 누가 하느냐가 다를 뿐입니다. 그리고 이것은 자본에 따라 달라지는 니즈가 아닙니다. 바쁜 직장인, 초보 구매자, 결정 피로에 시달리는 모든 사람이 원합니다.

"내가 좋아할 만한 것만 보여줘. 하지만 내가 고를게." AI가 바로 이 역할을 하는 겁니다.

쇼핑은 이렇게 바뀔수 있습니다.
- "겨울 코트 찾는데, 캐주얼하고 따뜻한 거"
- 에이전트가 쿠팡, 무신사, 29CM, 해외 사이트까지 탐색
- 내 체형, 과거 구매 데이터, 예산 고려해서 20개 추천
- 맘에 안든다고 하면 새로운 것 계속 추천
- 고른 다음 더 싸고 더 빠른 곳을 에이전트가 탐색

커머스만 그럴까요? 어플리케이션도 그렇습니다.
- "지난 분기 영업 실적 분석해줘"
- 에이전트가 세일즈포스, 구글 시트, 슬랙 메시지 탐색
- 필요한 API 가져오고, 데이터 정리하고, 워크플로우 만들고
- 나에게 맞는 UI 생성
- 다음에 필요하면 그대로 재사용

쇼핑은 매번 새로 탐색하지만, 업무는 한 번 만들면 계속 씁니다. "이번 분기도 똑같이"라고만 하면 끝입니다. 내가 세일즈포스 UI를 배울 필요가 없습니다. 어떤 API를 써야 하는지 몰라도 됩니다. 원하는 결과만 말하면, 에이전트가 알아서 방법을 찾습니다.

2. 이제는 컴퓨터가 사람에 맞춥니다.

지금까지의 UI는 생각해보면, 인간과 컴퓨터 사이의 번역기입니다. 인간은 자연어로 생각하지만, 컴퓨터는 정확한 명령만 실행할 수 있었습니다. 그래서 인간이 컴퓨터 방식에 맞춰왔습니다. 버튼을 클릭하고, 메뉴를 선택하고, 정해진 경로를 따라갑니다.

이제 방향이 역전됩니다. 컴퓨터가 인간에게 맞출 수 있습니다. 왜 지금일까요? 3가지가 동시에 갖춰졌기 때문입니다.

- 언어 이해: LLM이 의도와 맥락을 파악합니다. "따뜻한 겨울 코트"가 무슨 의미인지 실제로 이해합니다. 
- 추론: 복잡한 요청을 단계로 분해하고 "어떻게 해야 하는지" 계획을 세웁니다.
- 행동 능력: 여기는 아직 병목입니다. 이해하고 계획할 순 있었지만 실행은 잘 못합니다.

하지만, 최근 ChatGPT Agent SDK가 나오고, MCP 프로토콜이 생기고, Computer Use가 등장하면서 이제 시작됐습니다. API를 호출하고, 앱을 조작하고, 여러 서비스를 동시에 다룰 수 있게 될 것입니다. 곧, 3개가 모두 갖춰진 순간이 도래합니다.

3. 모든 사람이 VIP가 됩니다

쇼핑에서 탐색이 사라지는 것이 아니었습니다. 업무 툴의 복잡한 기능이 없어지는 것도 아니었습니다. 다만 내가 직접 하지 않을 뿐입니다. 백화점 VIP도 가끔은 직접 매장에 가서 구경합니다. 쇼핑 자체가 즐거울 때가 있습니다. 마찬가지로 에이전트를 통하지 않고 앱을 직접 열 때도 있을 것 입니다. 하지만 대부분의 경우는 에이전트를 통하게 되지 않을까요?

앱이 뭘 할 수 있는지 몰라도 괜찮습니다. 어떤 API를 써야 하는지 몰라도 괜찮습니다. 내 니즈만 말하면 됩니다.에이전트가 모든 가능성을 탐색하고, 내 맥락에 맞게 큐레이션하고, 맞춤혐 앱을 만들고, 맞춤형 UI를 제공하고, 나는 최종 선택만 하면 됩니다.

모든 인터페이스는 에이전트를 통하게 될 것 같습니다.
