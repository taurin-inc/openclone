AWS Summit Online Korea 2021 기조연설에서 **"한국 B2B SaaS가 글로벌 시장에서 경쟁하는 법"** 을 센드버드 사례로 공유했어요. 핵심 메시지는 세 가지.

첫째, **클라우드 인프라가 한국 스타트업의 글로벌 평준화**를 만들어줬어요. AWS·GCP 같은 글로벌 인프라를 쓰면 서울에서 빌드한 제품이 미국·유럽·동남아 어디서든 같은 레이턴시로 동작해요. 10~15년 전 기준으로는 상상할 수 없던 조건이죠. 이 조건 위에 **API-First 제품 설계**를 얹으면, 물리적으로 한국에 본사가 있어도 제품 자체는 글로벌 표준이 돼요.

둘째, **B2B SaaS 회사가 처음부터 신경써야 할 3가지 인프라 투자**:

1. **Observability** — 모니터링·로깅·트레이싱. 고객사가 장애 원인을 투명하게 확인할 수 있어야 엔터프라이즈 계약이 잡힙니다.
2. **Security & Compliance** — SOC 2, HIPAA, GDPR 같은 인증. 엔터프라이즈 영업의 전제조건.
3. **Scale Architecture** — 일일 수십억 메시지를 처리할 수 있는 이벤트 기반 아키텍처. 트래픽 폭발 대응이 B2B API 신뢰의 기반.

셋째, **"글로벌 표준 개발 문화"가 한국 스타트업의 경쟁력**이에요. 영어 문서화, Pull Request 기반 리뷰, 원격 협업 툴체인, 인시던트 포스트모템 문화. 이게 제품의 Global Composition을 만드는 기저 문화예요.

핵심 포인트:

- **클라우드 인프라가 한국 B2B SaaS의 글로벌 평준화**를 완성 — 지리적 거리보다 API 설계가 더 중요해진 시대.
- B2B SaaS 기본 인프라 투자 3대장 = **Observability · Security/Compliance · Scale Architecture**.
- **글로벌 표준 개발 문화** (영문 문서, PR 리뷰, 포스트모템)가 Global Composition의 기저.
- 센드버드는 이벤트 기반 아키텍처로 일일 수십억 메시지 처리.
- 엔터프라이즈 계약의 전제조건 = SOC 2 / HIPAA / GDPR 인증.
