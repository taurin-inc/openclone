# 2026 팁스(TIPS) 새로 바뀐 점 총정리 | 꼭 알아야 할 8가지 변경 사항

**Author:** Kakao Ventures  
**Date:** Feb 05, 2026

2026년 팁스(TIPS), 뭐가 달라졌을까요? 접수 방식부터 지원 금액, 가점 체계까지 주요 변경 사항 8가지를 정리했습니다. 팁스 신청을 준비 중이라면 꼭 확인해 보세요!

---

지난 1월, 중기부에서 2026년 팁스(TIPS) 창업기업 지원 계획을 발표했습니다. 접수 방식부터 지원 금액, 가점 체계까지 작년 대비 꽤 많은 부분이 달라졌는데요. 이번 글에서는 **2026년 팁스의 주요 변경 사항**을 하나씩 정리해 보았습니다.

## 1. 접수 방식 : 수시 → 분기별 접수

가장 먼저 눈에 띄는 변화는 **R&D 프로그램의 접수 방식**이에요. 2025년까지는 수시로 신청할 수 있었지만, 2026년부터는 **정해진 분기에만 접수**가 가능합니다.

| 구분 | 2025년 | 2026년 |
| :--- | :--- | :--- |
| R&D | 수시 접수 | **1·2·3분기 각 1회 (총 3회)** |
| 비R&D | 1분기 1회 접수 | 1분기 1회 접수 (동일) |

## 2. 일반트랙 지원금 상향 : 5억 → 8억

**일반트랙 R&D 지원금이 5억 원에서 8억원으로 상향**되었습니다. 사업기간은 동일하게 **최대 2년**이에요.

| 트랙 | 2025년 | 2026년 |
| :--- | :--- | :--- |
| 일반트랙 | 최대 5억 원 | **최대 8억 원** |
| 딥테크트랙 | 최대 15억 원 | 최대 15억 원 (동일) |

## 3. 글로벌 트랙 : 2026년 신규 모집 없음

2026년부터 기존 팁스 글로벌트랙은 폐지되고, **'글로벌팁스 R&D' 사업과 통합**되어 운영됩니다. 글로벌 진출을 준비 중이시라면 글로벌팁스 R&D 공고를 눈여겨보시면 좋을 것 같아요.

| 구분 | 2025년 | 2026년 |
| :--- | :--- | :--- |
| 글로벌트랙 | 최대 12억원 | **폐지 (신규 모집 없음)** |

## 4. 일반트랙 투자 요건 변경 : 수도권 2억 원, 비수도권 1억 원

팁스에 지원하려면 먼저 **팁스 운영사로부터 투자**를 받아야 하는데요. 이 **최소 투자금액 기준**이 바뀌었습니다.

| 구분 | 2025년 | 2026년 |
| :--- | :--- | :--- |
| 수도권 기업 | 1~2억원 | **2억원 이상** |
| 비수도권 기업 | 1~2억원 | **1억원 이상** |

**수도권 기업**은 투자 요건이 강화되었고, **비수도권 기업**은 완화되었습니다.

## 5. 신산업 분야 업력 기준 완화 : 창업 7년 → 10년 이내

팁스 일반트랙은 기본적으로 **창업 7년 이내 기업**이 지원할 수 있는데요. 2026년부터는 **신산업 분야 창업 기업**의 경우, **창업 후 10년 이내**까지 지원이 가능해졌습니다.

| 분야 | 2025년 | 2026년 |
| :--- | :--- | :--- |
| 일반 | 창업 7년 이내 | 창업 7년 이내 (동일) |
| 신산업 | 창업 7년 이내 | **창업 10년 이내** |

**신산업 분야:** AI, 반도체, 양자·보안, 로봇, 모빌리티, 생명·신약 등이 해당됩니다.

## 6. 딥테크트랙, 완전히 달라졌습니다!

딥테크트랙은 올해 지원 대상, 대상 분야, 평가 방식에서 큰 변화가 있었습니다.

### 1️⃣ 지원 대상이 바뀌었어요

| 구분 | 2025년 | 2026년 |
| :--- | :--- | :--- |
| 대상 | 팁스 수행 이력이 없는 기업 | **일반트랙 '완료' 판정 기업** |
| 투자 요건 | 3억원 이상 | **후속투자 5억원 이상** (운영사가 아닌 투자사도 OK) |
| 매출 기준 | 20억원 미만 | **20억원 이상도 가능** |
| 퇴직연금 | 해당 없음 | **🆕 도입 필수** |

2026년부터는 **일반트랙을 완료한 기업만** 딥테크트랙에 지원할 수 있으며, **후속투자 5억원 이상** 유치가 필수입니다. 또한 **퇴직연금제도 도입이 필수**가 되었습니다.

### 2️⃣ 대상 분야가 확대됐어요
기존 ‘10대 초격차 분야’에서 **‘12대 신산업 분야’**로 늘어났습니다.
*   AI, 반도체, 양자·보안, 로봇, 모빌리티
*   생명·신약, 헬스케어, 콘텐츠
*   방산·우주항공·해양, 친환경, 에너지·핵융합, 센서·공정

### 3️⃣ 평가 방식이 간소화됐어요

| 구분 | 2025년 | 2026년 |
| :--- | :--- | :--- |
| 평가 횟수 | 기술성·사업성 각각 (총 2회) | **통합 1회** |

## 7. 가점 항목이 늘어난 일반트랙

최대 가점이 3점에서 **5점으로 확대**되었고, 새로운 항목이 추가되었습니다.

### 2026년 팁스 일반트랙 가점 항목
| 항목 | 가점 | 비고 |
| :--- | :--- | :--- |
| 비수도권 본사 소재 | **최대 3점** | 선정물량 50% 초과 시 2점 |
| 비수도권 공장 소재 | 2점 | 공장등록증명서 필요 |
| ESG 분야 영위 | **최대 2점** | **🆕** 신설 |
| 퇴직연금제도 도입 | **1점** | **🆕** 신설 |
| 벤처기업·이노비즈 인증 | **최대 1점** | **🆕** 신설 |
| 내일채움공제 가입 | 1점 | (기존과 동일) |
| 프리팁스 '성공' 판정 | 2점 | (기존과 동일) |

## 8. 비R&D 연계사업 평가 간소화 : 대면평가 → 서류평가

창업사업화·해외마케팅 등 비R&D 연계사업의 평가 방식이 발표 부담을 줄이기 위해 서류평가로 변경되었습니다.

| 구분 | 2025년 | 2026년 |
| :--- | :--- | :--- |
| 평가 방식 | 대면평가 | **서류평가** |

*주의: 2026년 팁스 R&D에 새로 선정된 기업은 비R&D 연계사업에 당해년도 참여가 불가능하며 2027년부터 신청 가능합니다.*

## 한눈에 보는 2026년 팁스 변경사항

| 항목 | 2025년 | 2026년 |
| :--- | :--- | :--- |
| R&D 접수 방식 | 수시 | **분기별 (연 3회)** |
| 일반트랙 지원금 | 최대 5억원 | **최대 8억원** |
| 수도권 투자 요건 | 1~2억원 | **2억원 이상** |
| 비수도권 투자 요건 | 1~2억원 | **1억원 이상** |
| 신산업 업력 기준 | 7년 | **10년** |
| 딥테크트랙 대상 | 신규 기업 | **일반트랙 완료 기업** |
| 딥테크트랙 투자 요건 | 3억원 | **후속투자 5억원** |
| 일반트랙 최대 가점 | 3점 | **5점** |
| 비R&D 평가 방식 | 대면 | **서류** |

## 마무리하며

2026년 팁스는 지역 균형 발전과 신산업/딥테크 분야의 연속성 있는 지원에 초점을 맞춘 것으로 보입니다. 새로운 기준에 맞춰 철저히 준비하시어 좋은 결과 있으시길 응원합니다.
